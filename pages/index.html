Version

Due to the nature of the permaweb, if you open this page through an older manifest link, you’d only see the page as it was when the manifest is made. To see updated files, you’d have to find the latest manifest link, or through a traditional web page. There are three ways for you to find the latest version of the blog:

  1. Via xenomancy gateway: sba.xenomancy.id

    This method is preferred.

  2. Via a xenomancy clip page: clip.xenomancy.id

    This method might not be supported in the future. I suggest you to always use the first method. To ensure compatibility, instead of deleting it entirely, once I depreciate this methods, the link would redirect to sba.xenomancy.id.

  3. Find the latest version of the manifest file here. This method should never fail. Click on xenostation manifest file, then select the preview button.
